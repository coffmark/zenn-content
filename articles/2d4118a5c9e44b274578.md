## API GatewayでWebSocketの接続確認を行う

**インストール**
`npm install -g wscat`


**wscatコマンドの実行**
`wss://{api-id}.execute-api.{region}.amazonaws.com/{stageName}`

# 写真の追加

**APIをテストするには次のようなメッセージを入力する必要があります。**

`{"{jsonpath-expression}" : "{route-key}"}`

example

`{"action": "action1"}`

`{"message": "test response body"}`
